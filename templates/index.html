Container

List all containers:

curl -X GET "localhost:5000/container?database="

Create container:

curl -X POST "localhost:5000/container?container=&database=&partition_key="

Delete a container:

curl -X DELETE "localhost:5000/container?container=&database="

Item

Read an item:

curl -X GET "localhost:5000/item?id=&container=&database=&partition_key="

Upsert an item:

curl -X POST "localhost:5000/item?container=&database=" -d "{\"id\": \"1234\", \"property\": \"abcd\"}" -H "Content-Type: application/json"

Delete an item:

curl -X DELETE "localhost:5000/item?id=&container=&database=&partition_key="

Replace an item:

curl -X PUT "localhost:5000/item?container=&database=&id=" -d "{\"property\": \"abc\"}" -H "Content-Type: application/json"
